SQL ORDER BY CLAUSE


The ORDER BY clause in SQL is used to sort the result set of a query in ascending or descending order based on one or more columns. It is commonly used to arrange the rows returned by a SELECT statement in a specified order.

The ORDER BY clause is essential for organizing data and is frequently used to present data in a more meaningful and user-friendly way.

Syntax: The basic syntax of the SELECT statement is as follows:

Syntax: Copy
SELECT column1, column2, ... 
FROM table_name 
ORDER BY column1, column2 [ASC | DESC];

Note: You can specify one or more columns by which you want to sort the result set. The optional `ASC` keyword stands for "ascending" (default), and the `DESC` keyword stands for "descending." if column-1 data is same then it will check column-2 to order.


Table Name: employees:
id first_name last_name salary
1 John Doe 50000
2 Jane Smith 60000
3 Robert Johnson 55000
4 Emily Davis 62000


1. ASC To retrieve the data from the 'employees' table sorted by the 'salary' column in ascending order, you would use the following SQL query:

Example: Copy
SELECT id, first_name, last_name, salary 
FROM employees 
ORDER BY salary;


Result: This query will return the result set sorted by the 'salary' column in ascending order:
id first_name last_name salary
1 John Doe 50000
3 Robert Johnson 55000
2 Jane Smith 60000
4 Emily Davis 62000


2. DESC If you want to sort the data in descending order by salary, you can use the `DESC` keyword:

Example: Copy
SELECT id, first_name, last_name, salary 
FROM employees 
ORDER BY salary DESC;


Result: This query will return the result set sorted by the `salary` column in descending order:
id first_name last_name salary
4 Emily Davis 62000
2 Jane Smith 60000
3 Robert Johnson 55000
1 John Doe 50000


3. complex sorting You can also use the ORDER BY clause with multiple columns to perform more complex sorting. For example, to sort by `last_name` in ascending order and then by `first_name` in ascending order, you can use the following query:

Example: Copy
SELECT id, first_name, last_name, salary 
FROM employees 
ORDER BY last_name, first_name;

To summarize: The ORDER BY clause is versatile and crucial for controlling the order of your query results, making it easier to present and analyze your data effectively.
