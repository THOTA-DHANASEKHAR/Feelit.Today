SQL LOGICAL OPERATORS

SQL logical operators are used to combine multiple conditions in SQL queries to create more complex and flexible search criteria. They allow you to specify conditions that involve multiple expressions or columns and are typically used in the `WHERE` clause of SQL statements. The common SQL logical operators include:


Operator Description Example
AND It returns `TRUE` only if all the conditions are `TRUE`. SELECT * FROM employees WHERE age >= 30 AND salary >= 50000;
OR It returns `TRUE` if at least one of the conditions is `TRUE`. SELECT * FROM products WHERE category = 'Electronics' OR stock_quantity > 100;
NOT This operator is used to negate a condition, making a `TRUE` condition `FALSE` and vice versa. SELECT * FROM customers WHERE NOT country = 'USA';

These logical operators are often used to create more sophisticated filtering conditions in SQL queries. Here are some examples of how they can be used:


Table Name: employees
id name salary hire_dat
1 Alice 50000 2023-01-01
2 Bob 60000 2023-01-02
3 Charlie 55000 2023-01-03
4 David 70000 2023-01-04


1. AND Operator: The `AND` operator is used to retrieve rows that satisfy both conditions.

Example: Copy 
SELECT * 
FROM employees 
WHERE 
salary > 55000 
AND 
hire_date >= '2023-01-03';

Result:
id name salary hire_dat
4 David 70000 2023-01-04

This query retrieves employees who have a salary greater than 55000 and were hired on or after '2023-01-03'.



2. OR Operator: The `OR` operator is used to retrieve rows that satisfy either of the conditions.

Example: Copy 
SELECT * 
FROM employees 
WHERE salary > 60000 
OR 
hire_date < '2023-01-02';

Result:
id name salary hire_dat
1 Alice 50000 2023-01-01
2 Bob 60000 2023-01-02
4 David 70000 2023-01-04

This query retrieves employees who have a salary greater than 60000 or were hired before '2023-01-02'.



3. NOT Operator The `NOT` operator is used to negate a condition.

Example: Copy 
SELECT * 
FROM employees 
WHERE 
NOT salary == 60000;

Result:
id name salary hire_dat
1 Alice 50000 2023-01-0
3 Charlie 55000 2023-01-0
4 David 70000 2023-01-03

This query retrieves employees who do not have a salary equal to 60000.



These logical operators allow you to create more complex conditions in your SQL queries, enabling you to filter and retrieve data based on various criteria. They are particularly useful when you need to combine multiple conditions to express the desired logic for your query.
