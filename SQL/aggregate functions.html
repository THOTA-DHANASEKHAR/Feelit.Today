SQL AGGREGATE FUNCTIONS


Aggregate functions in SQL are used to perform calculations on sets of values and return a single value as a result. Common aggregate functions include COUNT, SUM, AVG, MIN, and MAX.

These functions are often used in conjunction with the `GROUP BY` clause to summarize data for groups or categories. Let's explore these aggregate functions with an example table.

Suppose you have a table named `sales` that stores information about sales transactions, including `product_id`, `quantity`, and `amount`. Here's a simplified representation of the table:

Example: Copy
CREATE TABLE sales ( 
            sale_id INT PRIMARY KEY, 
            product_id INT, 
            quantity INT, 
            amount DECIMAL(10, 2) 
        ); 
        INSERT INTO sales (
            sale_id, 
            product_id, 
            quantity, 
            amount
        ) 
        VALUES (1, 101, 5, 50.00), 
               (2, 102, 3, 30.00), 
               (3, 101, 2, 20.00), 
               (4, 103, 4, 40.00), 
               (5, 102, 6, 60.00);

Now, let's look at some common aggregate functions using this `sales` table:



1. COUNT Function: The COUNT function is used to count the number of rows in a result set. For example, to count the number of sales transactions, you can use:

Example: Copy
SELECT COUNT(*) AS total_sales 
FROM sales;

Result:
total_sales
5


2. SUM Function: The `SUM` function is used to calculate the sum of values in a column. To find the total sales amount, you can use:

Example: Copy
SELECT SUM(amount) AS total_amount 
FROM sales;

Result:
total_amount
200.00


3. AVG Function:The `AVG` function calculates the average of values in a column. To find the average quantity of products sold, you can use:

Example: Copy
SELECT AVG(quantity) AS average_quantity 
FROM sales;

Result:
average_quantity
4.00


4. MIN Function:The `MIN` function returns the minimum value in a column. To find the minimum sale amount, you can use:

Example: Copy
SELECT MIN(amount) AS min_amount 
FROM sales;

Result:
min_amount
20.00


5. MAX Function: The MAX function returns the maximum value in a column. To find the maximum sale quantity, you can use:

Example: Copy
SELECT MAX(quantity) AS max_quantity 
FROM sales;

Result:
max_quantity
6



6. GROUP BY with Aggregate Functions: You can also use these aggregate functions in combination with the `GROUP BY` clause to summarize data for groups or categories. For example, to calculate the total sales amount for each product, you can use:

Example: Copy
SELECT product_id, SUM(amount) AS total_amount 
FROM sales 
GROUP BY product_id;

Result:
product_id total_amount
101 70.00
102 90.00
103 40.00


To summarize: These are some of the common aggregate functions in SQL, along with examples of how to use them with a sample table. Aggregate functions are powerful tools for summarizing and analyzing data in a database, providing insights into your data for reporting and decision-making purposes.
