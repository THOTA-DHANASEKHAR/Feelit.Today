SQL DISTINCT


In SQL, the DISTINCT keyword is used in the 'SELECT' statement to retrieve unique values from a specified column or set of columns.It eliminates duplicate values from the result set, returning only distinct values.


Syntax: The basic syntax of the DISTINCT statement is as follows:

Syntax: Copy
SELECT 
DISTINCT column1, column2, ... 
FROM table_name;

Table Name: employees:
ID FirstName LastName
1 John Smith
2 Jane Doe
3 John Smith
4 Emily Johnson


1. DISTINCT : If you want to retrieve the distinct first names of the employees, you can use the `DISTINCT` keyword:

Example: Copy
SELECT 
DISTINCT FirstName 
FROM employees;

Result:
FirstName
John
Jane
Emily

As you can see, the DISTINCT keyword ensures that only unique values are returned. In this case, it removes the duplicate "John" entry from the result set.


To summarize: The DISTINCT keyword is helpful when you want to retrieve a list of unique values from one or more columns in a table. It is commonly used to identify unique values, create lists of options for dropdowns or filters, and eliminate redundancy in query results.
