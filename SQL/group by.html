SQL GROUP BY CLAUSE


The GROUP BY clause in SQL is used to group rows from a table based on the values in one or more columns. It is often used in conjunction with aggregate functions, such as SUM, COUNT, AVG, MIN, and MAX to perform calculations on groups of rows, rather than on individual rows.

The GROUP BY clause is particularly useful for summarizing data and generating reports that require data to be grouped and analyzed in categories or sets.


Syntax: The basic syntax of the GROUP BY statement is as follows:

Syntax: Copy
SELECT column1, aggregate_function(column) as column2, ... 
FROM table_name 
WHERE condition
GROUP BY column1, column2, ...;

1. Group calculation: You can use GROUP BY to perform Aggregations for each group of values.

If you want to find the total salary spent on each department, you can use the `GROUP BY` clause with the SUM() aggregate function and to find total employees in each depertment can use COUNT() aggregate function. you can also use WHERE condition if required

Example: Copy
SELECT dept_id, 
sum(salary) as total_salary, 
COUNT(*) as employee_count 
FROM employees
GROUP BY dept_id;

Result:
dept_id total_salary employee_count
1 105000 2
2 130000 2

To summarize: The GROUP BY clause is a powerful feature in SQL for summarizing and analyzing data by grouping it into categories. It is commonly used in reporting and data analysis to obtain insights and summaries of data.
