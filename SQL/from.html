from

SQL FROM STATEMENT


The FROM clause in SQL is used to specify the source table or tables from which you want to retrieve data. It is an essential part of the SELECT statement and is used to identify the tables involved in the query.


Syntax: The basic syntax of the FROM clause is as follows:

Syntax: Copy
-- Select from single table
SELECT column1, column2, ... 
FROM table_name;

-- OR
SELECT column1, column2, ...
FROM [database_nmae].[table_name];

-- Multiple Tables
SELECT t1.column1, t2.column2, t3.column2, ... 
FROM table1 t1
JOIN table2 t2 ON t1.column == t2.column
JOIN table3 t3 ON t2.column == t3.column;

1. Selecting from 1 Table: You can use FROM clause for selecting data from specific table.

Example: Copy
SELECT id, name, salary 
FROM employees;

Result:
id name salary
1 Alice 50000
2 Bob 60000
3 Charlie 55000
4 David 70000


2. Selecting data from 2 tables: You can use from clause with join for selecting data from multiple tables

Example: Copy 
SELECT e.name, d.dept_name 
FROM employees 
JOIN departments ON e.dept_id = d.dept_id;

-- OR 

SELECT employees.name, departments.dept_name 
FROM employees 
JOIN departments ON employees.dept_id = departments.dept_id;

Result:
name dept_name
Alice HR
Bob Finance
Charlie HR
David Finance

To summarize: The FROM clause is fundamental in SQL queries as it identifies the data source and allows you to perform operations on the specified tables. It is often used in conjunction with other clauses like `JOIN` to retrieve data from multiple tables.
