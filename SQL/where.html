SQL WHERE CLAUSE


The WHERE clause in SQL is used to filter rows from a table based on specified conditions. It is an essential component of SQL statements, such as SELECT, UPDATE, DELETE, and MERGE allowing you to define criteria that determine which rows are included in the result set or affected by an operation.

The WHERE clause is used to apply conditions to one or more columns in a table, and it returns only the rows that satisfy those conditions.


Syntax: The basic syntax of the WHERE statement is as follows:

Example: Copy
SELECT column1, column2, ... 
FROM table_name 
WHERE condition;


Table Name: students:
ID name grade age
1 John A 18
2 Sam B 19
3 Robert C 20
4 Emily F 19


1. If you want to retrieve the names of all students who are 19 years old, you can use the following SELECT statement with a WHERE clause:

Example: Copy
SELECT name, age 
FROM students 
WHERE age = 19;

Result:
name age
Sam 19
Emily 19


2. Complex Conditions: The WHERE clause allows you to create complex conditions using logical operators ( AND, OR, NOT ) and parentheses to group conditions.

For example, to retrieve students who are 19 years old and have a grade of 'B' or 'C' you can use the 'OR' operator:

Example: Copy
SELECT name 
FROM students 
WHERE age = 19 AND (grade = 'B' OR grade = 'C');

Result:
name age
Sam 19

To summarize: the WHERE clause is a fundamental tool for filtering and retrieving specific data from a database. It is commonly used to restrict the results of SQL queries to match specific criteria and is essential for data analysis, reporting, and data manipulation tasks.
